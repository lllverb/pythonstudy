# 背景

## 機能

- 画像のスクレイピング
  1. scrape.py を実行
  2. 「検索キーワード」「拡張子」「保存したい枚数」「保存したいディレクトリ」を入力
  3. 上記の情報を持って Google 画像検索から画像を保存。
  4. 保存した画像をナンバーを付けてリネーム。
  5. opencv を用いて保存した画像の中から顔と認識できるものを新しいディレクトリに保存。その際、切り取った元画像のナンバーを使用して命名。
